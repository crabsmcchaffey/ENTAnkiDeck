# The Crabs ENT Learning Ecosystem

This deck is part of a larger collection of free educational resources for Otolaryngology trainees.

## 🦀 CrabsMcChaffey Lightyear ENT Deck

A comprehensive ENT Anki deck designed for medical students, sub-interns, and residents.

## 🥋 Crabs Staging Dojo

An interactive Head & Neck cancer staging trainer built to help learners master AJCC 8th Edition TNM staging through deliberate practice.

Current staging modules include:

* Oral Cavity
* Oropharynx (HPV+)
* Oropharynx (HPV−)
* Larynx
* Hypopharynx
* Nasopharynx

🌐 **Crabs Staging Dojo:**
https://oral-cavity-staging-ninja.vercel.app/quiz

The Staging Dojo is designed to complement the staging content contained within this deck and is particularly useful for residents preparing for OITE, board examinations, and multidisciplinary tumor board discussions.

Together, the deck and Dojo provide both long-term knowledge retention and active clinical application.

# Quick Start

## Medical Student (ENT Elective)

1. Read AAO Primary Care Otolaryngology
2. Unsuspend corresponding cards
3. Complete cards by subspecialty

Recommended tags:

```text
Crabs::Elective
```

## Medical Student (Sub-I / Away Rotation)

1. Read ENT Secrets
2. Unsuspend specialty-specific cards
3. Supplement with anatomy cards
4. Review procedure cards before cases

Recommended tags:

```text
Crabs::Sub-I
Crabs::Specialty::*
Crabs::extra_anatomy
```

## Residents

1. Read the corresponding textbook chapters
2. Complete the associated cards
3. Use procedure cards before operative cases
4. Use staging and oncology sections for board preparation

Recommended tags:

```text
Crabs::Residency
Crabs::Procedures
```

# Tagging Philosophy

The deck is heavily organized using Anki tags.

Examples include:

```text
Crabs::Specialty::Otology
Crabs::Specialty::Rhinology
Crabs::Specialty::HeadandNeck

Crabs::Procedures::Thyroidectomy
Crabs::Procedures::Tracheostomy
Crabs::Procedures::Parotidectomy

Crabs::Elective
Crabs::Sub-I
Crabs::Residency

Crabs::extra_anatomy
Crabs::Pimped
```

This allows users to tailor the deck to their level of training and current clinical rotation.

# Recommended Study Strategy

## Medical Students

Read ENT Secrets and complete the corresponding cards.

## Sub-Interns

Read ENT Secrets and Pasha, then complete the corresponding cards and procedure sections.

## Residents

Read the relevant textbook chapter, complete the cards, and review procedure-specific material before cases.

Remember:

> Anki is a retention tool, not a learning tool.

The deck is intended to supplement—not replace—textbooks, primary literature, operative atlases, and clinical experience.

When a new version is released:

1. Back up your Anki collection.

3. Import the updated `.apkg`.
4. Ensure import settings preserve review history and scheduling information.
